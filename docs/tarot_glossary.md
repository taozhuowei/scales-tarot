## 1. 基础概念

1. 塔罗牌（Tarot）：一套共 78 张的占卜用牌，分为大阿卡纳 22 张与小阿卡纳 56 张。
2. 正位（Upright）：卡牌抽出后图案方向朝向解读者，通常代表能量自然、正向流动。
3. 逆位（Reversed）：卡牌抽出后图案方向上下颠倒，通常代表能量受阻、内在化或反向呈现。
4. 洗牌（Shuffle）：将整副塔罗牌打乱顺序，重置牌序。
5. 切牌（Cut）：在洗牌完成后，将牌堆从中间分开再合并的仪式动作，常带有提问者的能量介入意义。
6. 抽牌（Draw）：从牌堆中抽出一张或多张牌作为占卜结果。
7. 翻牌（Flip）：将抽出的卡牌从背面翻转为正面，揭示牌面与正逆位状态。
8. 占卜（Divination）：通过塔罗牌进行问题解答与方向探查的完整过程。
9. 解读（Reading）：基于卡牌牌面、正逆位与牌阵位置进行的意义阐释。

## 4. 四元素（Four Elements）

塔罗的四花色对应西方神秘学体系中的四大元素，每种元素承载特定的能量属性。

1. 火（Fire）— 对应权杖：行动、激情、创造、意志、灵感与精神冲劲。
2. 水（Water）— 对应圣杯：情感、关系、直觉、潜意识、爱与共情。
3. 风（Air）— 对应宝剑：思考、沟通、理性、判断、冲突与决断。
4. 土（Earth）— 对应钱币：物质、财富、身体、稳定、工作与现实根基。

## 5. 宫廷牌（Court Cards）

每个花色含四张宫廷牌，对应不同的角色阶段与心理原型。

1. 侍从（Page）：学习者、初探者、信使。代表事物刚萌芽的阶段，象征好奇心、学习意愿与新机会的预兆。
2. 骑士（Knight）：行动者、追寻者。代表能量充沛、向外推进的阶段，象征行动力、冲刺与极端化的元素特质。
3. 王后（Queen）：成熟内化的女性原型。代表向内深耕、滋养与理解，象征花色能量的内在掌握与共情表达。
4. 国王（King）：成熟外化的男性原型。代表对外展现的权威与领导力，象征花色能量的稳定运用与对外影响力。

## 6. 正逆位规则

正逆位是塔罗解读的基础变量之一，同一张牌在正位与逆位下含义存在差异。

1. 正位（Upright）：图案方向朝向解读者。通常表示该牌核心能量自然、外显、正向流动；解读以牌面主流意义为主。
2. 逆位（Reversed）：图案方向上下颠倒。通常有以下几种解读取向（具体取决于流派与上下文）：
   - 能量受阻：正位能量被压抑、延迟或卡住。
   - 内在化：能量未在外显层面表达，而是转向内部体验。
   - 反向呈现：核心意义出现反转或扭曲。
   - 过度或不足：正位特质表现过头或严重缺失。
3. 解读时应结合牌阵位置、提问情境与周围牌组综合判断，单凭正逆位不足以下定论。
